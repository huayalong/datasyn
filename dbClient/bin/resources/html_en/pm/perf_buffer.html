System cache to provide the following data:

*. Buffer Read: Read a few pieces of the buffer

*. Buffer write: Number of write buffer block

*. DataFile Read: Caused by the buffer zone data files to read the real number of blocks

*. DataFile Write: Caused by the buffer zone data files to write the real number of blocks

*. Buffer hit rate: Read to the buffer hit probability

*. Buffer Size: The buffer zone has been in use for the total number of the page
