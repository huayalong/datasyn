System I / O to provide the following data:

*. Reading data file: Block read data files of a few

*. Data file to write: Data files into the block a few

*. Reading the log file: Log file to read a few pieces

*. Log file to write: Log file number of the write block

*. Reading control file: Control block the number of documents to read

*. Control file to write: Control the number of blocks to write documents

*. Buffer read and write: Read and write buffer block number
