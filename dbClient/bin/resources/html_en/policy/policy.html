Database user management is one of the core of database secure management. ShenTong database provides a mechanism for the effective management of database users.

The database users include two categories: the operating system user / group and the normal database users. In the management of these two categories the user hasn¡¯t any distinction. For each user, you can specify:
 
  • Username and Password:
  • whether the system administrator privileges
  • The default database schema used
  • Account validity
  • Account Lockout Policy
  • The user's session-level configuration information
 
The user's permissions can be divided into two categories, one is system permission, the other is permission of individual objects of system. Permissions on the system included creating a user, creating mode, starting / ending of the audit, the audit trail view in the database. System privilege can only be part of the system pre-defined role.
 
To learn more, please read the relevant sections of the system administrator manuals.
